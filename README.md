FontDumpTruck will produce .png files for every glyph/character in a selected font.

Run via *"fontdumptruck.exe -i"* to select from a list of all installed fonts on the machine.

Most fonts have many thousands of glyphs. FontDumpTruck will produce a .png file with each glyph for the entire unicode range, rather 
than just ASCII or ASCII extended. Running the app could take some time and will produce *a lot* of files.

Written years ago, recently updated with .net10 compatability and will likely see no further maintainence.
